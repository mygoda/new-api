API Provider -> 选择"**MiniMax**" 3. MiniMax Entrypoint -> `api.minimaxi.com` 4. MiniMax API Key 输入从 [MiniMax 开放平台](https://platform.minimaxi.com/user-center/payment/token-plan) (国际用户可访问 [MiniMax Developer Platform](https://platform.minimax.io/user-center/payment/token-plan)) 获取的 **API Key** 5. Model 选择 **MiniMax-M2.7** 6. 依次点击右上角的"Save"和"Done"按钮，保存配置 7. 开始使用"MiniMax-M2.7"
